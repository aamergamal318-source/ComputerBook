# computer-workbooks-pro

ثلاث دوسيات حاسوب مدرسية متدرجة حسب المستوى، وليست حسب الصف:

- الجزء الأول: مبتدئ
- الجزء الثاني: متوسط
- الجزء الثالث: متقدم

كل دوسية تحتوي 30 صفحة A4، وفيها شرح عملي، مساحات إجابة، تدريبات، ومهمات دورية للطلاب المبدعين.

## فتح ملفات HTML

- grade-7/index.html: الجزء الأول - مبتدئ
- grade-8/index.html: الجزء الثاني - متوسط
- grade-9/index.html: الجزء الثالث - متقدم

## توليد PDF

```bash
node generate-workbooks.js
node convert-to-pdf.js
```

## الملفات النهائية

- grade-7/grade-7.pdf
- grade-8/grade-8.pdf
- grade-9/grade-9.pdf

## ملاحظات للطباعة

الملفات مصممة للطباعة الملونة على A4. تحتوي الصفحات على مساحات إجابة يمكن للطلاب الكتابة عليها بعد الطباعة.
